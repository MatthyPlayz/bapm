What is BaPM ?

BaPM stands from BAsh Package Manager, or BaPM, and it allows you to download files
as a .bapmr file in a tar.gz archive.

What do .bapmr files do?

Usually, people using BaPM should be able to provide the extension to the thing you are downloading.
All you have to do is extract the tar.gz and rename the .bapmr extnesion to the extension they told you!
NOTE: DOES NOT WORK WITH ARCHIVES.
